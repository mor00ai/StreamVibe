Guida Definitiva: Pubblicare la tua PWA Musicale

Questa guida ti accompagna passo passo nella pubblicazione della tua PWA (Frontend + Backend). Quando separiamo il motore di ricerca di YouTube (il backend) dall'interfaccia visiva (il frontend), creiamo un'app vera e professionale senza blocchi.

1. Come funziona la nostra Architettura

  • GitHub: È la "cassaforte" del tuo codice. Sia il frontend che il backend vivono qui dentro, nella stessa cartella principale.
  • Render.com (Il Backend): Prende SOLO la cartella server da GitHub, esegue Node.js e bypassa YouTube per fornire lo streaming in background.
  • Netlify / GitHub Pages (Il Frontend): Prende i file visivi (index.html, app.js, style.css) da GitHub e crea il sito finale che installerai sul telefono.
Nota: Per funzionalità puramente legate a conti e registrazioni (utenti e password), nelle tue app future il servizio migliore sarà Firebase. Ma per lo streaming specifico da YouTube avevamo assolutamente bisogno di un server privato come Render, perché richiedeva "forza bruta" e calcoli lato server per decriptare e scaricare l'audio, cosa che Firebase (Essendo puramente un Database) non permette di fare.

2. GitHub: Il Caricamento Globale

L'obiettivo è portare tutto il progetto online su GitHub per renderlo accessibile dai servizi di hosting.

  1. Vai su GitHub.com e accedi.
  2. Clicca sul tasto verde New per creare un nuovo Repository. Dai un nome (es: pwa-musica-streamvibe).
  3. Clicca su Create repository.
  4. Apri la pagina del repository appena creato e clicca su "uploading an existing file".
  5. Dal tuo computer, apri la cartella locale dell'app.
  6. Seleziona tutti i file e cartelle in blocco (i file html, l'app.js, ma anche le intere cartelle server, e icons).
  7. Trascina tutto il blocco selezionato al centro della pagina web di GitHub.
  8. Attendi il caricamento e clicca su Commit changes (il bottone verde in basso).

3. Render.com: Accendere il Server (Backend)

Ora che i file sono online, diamo vita al motore Node.js.

  1. Vai su Render.com e registrati.
  2. Nella Dashboard, clicca su New + e scegli Web Service.
  3. Scegli "Build and deploy from a Git repository" e procedi.
  4. Collega il tuo account GitHub e spunta il repository appena creato. Clicca Connect.
  5. Opzioni di Configurazione (CRITICO):
    • Name: streamvibe-backend (o un nome a piacere).
    • Root Directory: Scrivi esplicitamente server.
    • Runtime: Node.
    • Build Command: Scrivi npm install.
    • Start Command: Scrivi node index.js.
    • Instance Type: Seleziona il piano Free (gratuito).
  6. Clicca Create Web Service in basso alla pagina.
  7. Attendi un paio di minuti. In alto a sinistra comparirà l'URL pubblico del tuo server, che dev'essere simile a https://streamvibe-backend.onrender.com.

4. Collegare il Server all'App (Aggiornamento di app.js)

L'app visiva ha bisogno di sapere qual è il "numero di telefono" del nuovo server su Render per chiedergli la musica e bypassare YouTube.

  1. Sul tuo computer, apri il file app.js.
  2. Cerca la riga della costante BACKEND_URL (solitamente riga 3), e sostituisci il vecchio http://localhost:3000 con il nuovo link appena copiato da Render.
    (Assicurati di incollarlo bene all'interno degli apici e non mettere la barra finale /).
  3. Torna sul tuo repository GitHub, dove avevi già caricato i file.
  4. Prendi il tuo file app.js modificato e trascinalo sopra la pagina GitHub, facendo di nuovo "Commit changes" in modo da sovrascriverlo. In questo modo Netlify riceverà l'app aggiornata con l'indirizzo vero!

5. Netlify: Mettere Online la PWA (Frontend)

Infine, mettiamo online la facciata dell'app, in modo da poter accedere al player dal telefono con un link pulito.

  1. Vai su Netlify.com e registrati usando l'account GitHub.
  2. Nella dashboard, clicca su Add new site e poi su Import an existing project.
  3. Seleziona Deploy with GitHub e scegli il tuo repository musicale.
  4. Nella schermata "Site Settings":
    • Base directory: Lascia completamente vuoto!
    • Build command: Lascia vuoto.
    • Publish directory: Lascia vuoto.
  5. Clicca su Deploy site.
  6. In pochi secondi Netlify genererà un link verde. Quello è il sito finale della tua PWA! Se lo apri dal cellulare ti basterà fare Condividi -> Aggiungi a Schermata Home per averla a tutto schermo esattamente come fosse scaricata dagli store!
